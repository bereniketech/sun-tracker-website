## Decisions Made

1. **Icon Generation**: Used Sharp (already available in Next.js) to convert SVG to PNG at 192x192 and 512x512 sizes, avoiding external services.

2. **Offline Caching Strategy**: Implemented localStorage-based caching in `suncalc-offline.ts` with 24-hour TTL for sun data, enabling offline functionality for previously viewed locations.

3. **Service Worker Approach**: Relied on next-pwa's auto-generated service worker with Workbox instead of manual configuration, ensuring production-ready caching strategies.

4. **Offline Page Design**: Created a user-friendly offline page showing last cached sun data with online/offline status indicators using Lucide icons.

5. **Manifest Configuration**: Set `display: "standalone"` for native app feel, `theme_color: "#f59e0b"` matching app branding, and included both narrow and wide screenshots.

6. **Type Safety**: Used type assertions (`as any`) for next-pwa config to work around version mismatch in type definitions between next-pwa's bundled Next.js types and the current project's types.
